algoritma untuk menghitung luas lingkaran menggunakan JavaScript:

Buat variabel r dan berikan nilai jari-jari lingkaran.
Hitung luas lingkaran dengan rumus π x r^2 dan simpan hasilnya di variabel luas.
Tampilkan hasil luas lingkaran dengan menggunakan fungsi console.log().
Berikut adalah contoh kode JavaScript yang mengimplementasikan algoritma:

// Menghitung Luas Lingkaran

// 1. Buat variabel r dan berikan nilai jari-jari lingkaran.
const r = 7;

// 2. Hitung luas lingkaran dengan rumus π x r^2 dan simpan hasilnya di variabel luas.
const luas = Math.PI * r ** 2;

// 3. Tampilkan hasil luas lingkaran dengan menggunakan fungsi console.log().
console.log(`Luas lingkaran dengan jari-jari ${r} adalah ${luas.toFixed(2)}`);


nilai luas akan di bulatkan menjadi 2 angka di belakang koma dengan menggunakan metode toFixed(). Nilai π dapat diakses melalui properti Math.PI yang disediakan oleh JavaScript.
